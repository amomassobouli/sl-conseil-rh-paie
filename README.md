## Déploiement

Site statique — déposez les 4 fichiers HTML sur n'importe quel hébergeur :

- **GitHub Pages** : Settings → Pages → Deploy from branch `main`
- **Netlify** : Drag & drop du dossier sur app.netlify.com
- **OVH / Hostinger** : Upload FTP classique
